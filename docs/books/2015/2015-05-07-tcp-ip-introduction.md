# 『基礎からわかるTCP/IPネットワークコンピューティング入門 第3版』を読んで

*2015-05-07 公開*

『[基礎からわかるTCP/IPネットワークコンピューティング入門 第3版](http://shop.ohmsha.co.jp/shopdetail/000000004145/)』を読んだ。

学部・大学院ではグラフ理論を勉強したり、研究したものの、ネットワークはずっと苦手意識を持っていた。
定評のある『マスタリングTCP/IP 入門編』すら通読できず、大学院でネットワーク基礎という講義も受けた覚えがあるが何一つ頭に入らなかった。
『基礎からわかるTCP/IPネットワークコンピューティング入門 第3版』はTwitterで名前だけ覚えていた。
仕事でネットワーク関連の仕事もあったりするので今の内に勉強しておこうと思い立ち、購入した。
私はよく本を購入するが積読ばかりであったが、この本は躓きも無く読むことが出来た。
最初の1章はある意味宣伝(TCP/IPは色んなところで使われているから勉強しようね)という内容であったが、第3章から計算機についての話から始まり、きちんと段階を踏んで説明をしてくれる。初めてネットワーク関連の本を通読することが出来た。
勿論、1回読んでも理解できるわけでもなく、何度も読んで理解するしかないが、今まで通しで読めなかったテーマが読めるようになるのはうれしい。
1つ気になったのは、参考文献が特になかったこと、であろうか。これを理解した後に読むとよい本があれば教えて欲しかった。『マスタリングTCP/IP』シリーズだろうか。
