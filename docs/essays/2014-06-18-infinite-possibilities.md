# 無限の可能性

*2014-06-18 公開*

人生には無限の可能性があるという。

僕はいわゆるIT企業に属している。
IT企業には、情報技術を駆使して世の中を便利にするという使命がある。
もしこの使命を狂信する上司が「我々の事業には無限の可能性がある」と口にしたとき、どう対処すべきだろうか。
人生における無限の可能性について答えるのは難しいが、計算機についてなら、ある程度は答えられるのではないだろうか。

まず、実現可能な事柄の集合を考える。
このとき、その集合の濃度はどうなっているのか。
次に、計算機によって実現可能な事柄の集合を考える。
こちらの濃度はどうだろうか。
計算機は結局、0と1からなる有限列しか扱えない。
つまり、どれほど高性能な計算機をいくつ用意しても、実現できる事柄は高々可算である。
一方、この世の中には数多の事象があり、僕はそれが非可算集合だと考えている。
そう、計算機がどれだけ頑張っても、解決できる事象は高々可算個にとどまるのである。

我々はこの事実を悲観すべきだろうか。
僕はそうは思わない。
むしろ、計算機の限界を知ることで、我々は驕らずに謙虚でいられる。
限界はある。では、その範囲内で何をすべきか。何ができるのか。

> ほら、よく言うだろ〜？人生には無限の可能性があるってさ〜。僕はそんなの信じてないんだ。いつだって選べる道は少なかった。時には道は1本しかなかった。その、少なかった可能性の中から自分で選んだ結果が僕をここまで連れてきた。だからこそ僕はその選んだ道を……選ばなくちゃならなかった道を大切にしたい。
