# 整理整頓すると仕事ができるようになる？

*2015-05-21 公開*

最近、私の周囲で「整理整頓をしましょう。整理整頓ができる人は仕事ができる。相関関係がある」という話をよく聞く。
このような正論に何か言うのもはばかられるが、何とも言葉にしがたい違和感を覚えている。
その違和感を言語化し、文章に起こすことで頭を整理したい。

「整理整頓ができる人は仕事ができる」という主張だが、これは順序が逆ではないかと思っている。
つまり、「仕事ができる人は整理整頓もできる」という「AならばB」という命題の逆、「整理整頓ができる人は仕事ができる」「BならばA」も真であると周囲が信じているからではないかと考える。

「仕事ができる」だとあまりにも曖昧なので、それを「物事を論理的に順序立てて考えられる」に置き換えた命題を考えてみる。
なるほど、確かに「物事を論理的に順序立てて考えられる」人なら、その能力を片付けにも生かして、きれいに整理整頓ができそうである。
一方、身の回りの整理整頓ができる人が「論理的に順序立てて考えられる」のかどうかは、果たして明らかだろうか。
「論理的に順序立てて考えられる」ほうが1段階難しいように感じる。帰納より演繹のほうが簡単そうに思えるのである。

尤も、命題論理の話に帰着させずとも、「整理整頓ができる人は仕事ができる」は「当たり前」という言葉を使わずに正当化できると思う。
会社なら大抵、上司なる存在がいて、何らかの査定を行う。査定をするためには評価をしなければならない。どうやって？
そこで、手っ取り早く「身の回りの整理整頓ができているか」に着目するのではなかろうか。
他人の評価は難しい事柄であるが、「身の回りの整理整頓ができているか」は見れば明らかにわかる。
そこから普段の成果物と照らし合わせて「整理整頓ができる人は仕事ができる」へと結びつける流れがあるのではないだろうか。
要は、妙な揚げ足を取られないように身の回りの整理整頓をしておきましょうね、という、この世を生き抜く知恵として理解することもできよう。
職場によっては、特にブルーカラーの現場では、整理整頓の出来が命取りになるかもしれない。
そして、意味もなく白いシャツを着せられ、首にネクタイを締めることを強要される私も、安直な評価から身を守るために身の回りを整理整頓するのである。
